## 良かった点
- なぜその判断に至ったのか、都度コメントで残してくれている。
- 使用上限まで達しても1か月後にはトークンが復活する。
- チャットGPTと見た目が酷似しており、チャットGPTの使用者であれば直感的に使いやすい。
## 困った点
- 時間が長い。(1h4m25s)※CodeX×アトラシアンの際はもう少し短かった\
  'hello world'出力程度で1時間かかると実用化に至れるか検討の余地あり。
- 要件ファイルが文字化けしているといわれた\
  私の環境では文字化けしていなかった。
- PRが指定したフォーマット通りに作成されていなかった。
- ローカルPCでのテストが失敗した。\
  原因はcompileSdk設定の誤りによるビルドエラー\
  さらなる原因はテストコードのコンパイルのみを試し、CodeXは実際にテストは行っていないこと？
## 改善したい点
- 成果物出力までの時間を短くしたい
- PRを指定したフォーマット通りに作成したい
- テストまで行ってほしい
- 作業が失敗した場合のロールバックルールを作成したい
## 次回試すこと
- 成果物出力までの時間を短くするためにどうすれば良いか検討すること
- 作業が失敗した場合のロールバックルールを作成すること
